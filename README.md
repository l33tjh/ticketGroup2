#### 메세지 큐를 이용한 티켓팅 사이트
---
메세지 큐를 이용한 티켓팅 사이트입니다.

회원가입 및 로그인을 진행한 뒤 예매하고자 하는 공연의 티켓팅을 진행합니다.

티켓팅 진행 시 발생되는 예매 데이터는 메세지 큐를 통해 처리되어 DB에 저장됩니다.
